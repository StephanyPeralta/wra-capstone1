# YouWize App

A YouTube-inspired app seamlessly integrated with the [YouTube API](https://developers.google.com/youtube). Search for any video, explore its details, and enjoy playbackâ€”all within one platform. Sign up or log in to save your favorite videos to a personalized 'Favorite List.' Try the light or dark mode interface for enhanced viewing.

## Usage

- The home page will show React videos as the default search.
- Find a video by typing a keyword in the search field.
- Click on the Video Card to reproduce the video.
- Sign Up / Log In to save videos in your Favorite List.
- Switch to light or dark mode.
